# Data Sources

All source materials used are from public-domain or government sources. No copyrighted or pirated materials are included.

## Constitution of India

- **Source**: Legislative Department, Ministry of Law and Justice
- **URL**: https://legislative.gov.in/constitution-of-india/
- **License**: Government publication (public domain)
- **Format**: PDF
- **Content**: All Parts, Articles, Schedules, and Amendments

## Central Bare Acts (India Code)

- **Source**: India Code Information System, Legislative Department
- **URL**: https://www.indiacode.nic.in
- **License**: Government publication (public domain)
- **Key Acts**:
  - Indian Penal Code 1860 (IPC / BNS)
  - Code of Criminal Procedure 1973 (CrPC / BNSS)
  - Indian Evidence Act 1872 (IEA / BSA)
  - Transfer of Property Act 1882
  - Contract Act 1872
  - Constitution of India (already listed above)
  - And all Central Acts accessible via India Code

## Supreme Court Judgments

- **Source**: Supreme Court of India
- **URL**: https://sci.gov.in/judgments
- **License**: Public domain (court records)
- **Coverage**: Landmark constitutional bench and division bench judgments

## High Court Judgments

- **Source**: Various High Court websites
- **License**: Public domain (court records)

## Data Processing

All documents are:
1. Downloaded from official government sources
2. Text extracted using PyMuPDF / OCR
3. Cleaned and normalized
4. Chunked by legal structure (Article/Section/Chapter)
5. Embedded and stored in local vector database

No data is sent to any external service. All processing is performed locally.
