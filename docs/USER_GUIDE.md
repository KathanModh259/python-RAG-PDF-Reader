# User Guide

## Getting Started

### Installation

1. Download the latest LegalAI-Setup.exe from the releases page
2. Run the installer and follow the setup wizard
3. Launch Legal AI from the Start Menu or desktop shortcut
4. On first run, the application will download the LLM model (~4.7GB)
5. Follow the first-run wizard to:
   - Accept the disclaimer
   - Select your model preferences
   - Download or import legal document corpora

### Importing Documents

1. Click "Import" or drag-and-drop files into the document library
2. Supported formats: PDF, DOCX, TXT, HTML, EPUB, MD, images (JPG, PNG, TIFF)
3. The system will automatically extract text and index the content
4. For scanned documents, OCR runs automatically

### Asking Questions

1. Type your question in the chat input box
2. Press Enter or click "Ask"
3. The system will:
   - Search indexed documents for relevant content
   - Rerank results for relevance
   - Generate an answer grounded in the retrieved documents
   - Display citations and source documents

### Query Modes

- **Standard**: Full legal analysis with citations
- **Layman**: Simplified explanation in plain language
- **Summary**: Concise summary of relevant content
- **Contract Review**: Risk flagging and clause analysis

## Features

### Legal Q&A
Ask questions about Indian law and get answers with specific Article/Section citations.

### Document Library
Manage your indexed documents. View, delete, or re-index documents.

### Citation Panel
View the exact sources used to generate each answer, with page numbers and document names.

### Settings
- Model selection and configuration
- GPU acceleration toggle
- Data directory management
- Encryption passphrase setup
- OCR engine selection

## Privacy

Legal AI runs entirely offline. No data leaves your computer. Documents and indexes are stored locally. Encryption is available for sensitive documents.

## Disclaimer

This tool provides informational assistance only and is not a substitute for professional legal advice. Always consult a qualified legal practitioner for legal matters.
